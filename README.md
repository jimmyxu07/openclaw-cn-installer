# OpenClaw 中文版 Windows 安装包

🐱‍💻 **开箱即用，只需输入 API Key**

## 下载
https://github.com/jimmyxu07/openclaw-cn-installer/releases

## 使用
1. 双击安装包
2. 选择模型，输入 API Key
3. 点击桌面图标启动

## 支持模型
- 阿里百炼 ⭐️推荐（最便宜：¥0.5/M tokens）
- Moonshot Kimi（免费额度）
- 智谱 GLM-5
- MiniMax-M2.5
- 阿里通义 Qwen
- 自定义中转站

## 系统要求
- Windows 10 (1903+) 或 Windows 11
- 4GB RAM（推荐 8GB）
- 500MB 磁盘空间
